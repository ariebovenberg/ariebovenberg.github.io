# Development

On MacOS:

```bash
brew install rbenv  # follow additional installation instructions
rbenv init
curl -fsSL https://github.com/rbenv/rbenv-installer/raw/main/bin/rbenv-doctor | bash
rbenv install 3.0.3

# restart the shell

# in the repo
rbenv local 3.0.3

gem install bundler jekyll
```

# Updating

```bash
bundle update github-pages
```
